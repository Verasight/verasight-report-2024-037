# Methodology

Methodology summary for Verasight MPSA Omnibus Survey #2024-037.

## Survey

- Sponsor: Verasight
- Field dates: 2024-04-10 to 2024-04-15
- Population: US adults age 18+
- Mode: online panel (Verasight Community)
- Questions: 78
- Base size: N=1000.

## Weighting

- Weight variable: module-specific weights: weight_m1 for module 1, weight_m2 for module 2
- CPS variables: age, race/ethnicity, sex, income, education, region, metropolitan status
- Partisanship source: Pew Research Center NPORS benchmarking surveys, three-year running average
